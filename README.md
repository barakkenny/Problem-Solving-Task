# Cashier Assignment

This project provides functions to process an array of student objects to calculate their average grades, find the top student, and sort students by their average grades.

## Files

- `cashier.js`: Contains the functions to calculate average grades, find the top student, and sort students by average grades.

## Functions

### `calculateAverageGrades(students)`

Takes an array of student objects and returns a new array of objects. Each object contains the `id`, `name`, and `averageGrade` of each student.

### `findTopStudent(students)`

Takes the array of student objects and returns the object representing the student with the highest average grade.

### `sortStudentsByGrade(students)`

Takes the array of student objects and returns a new array sorted by the average grade in descending order.

## How to Run

1. Ensure you have Node.js installed on your system.
2. Clone this repository or download the files.
3. Navigate to the directory containing the files in your terminal.
4. Run the following command to execute the `cashier.js` file:

```sh
node cashier.js
